# Randomizer

#### Looking to cure boredom? Time to hit the Randomizer!

https://the-randomizer.herokuapp.com/

======

### Configuration

* git clone `https://github.com/anniehck/randomizer.git`

* bundle install

* npm install

* rake db:create

* rake db:migrate

* rails server

* npm start

======

### Technologies

* Ruby 2.3.1
* React.js
* Rails 4.2.6
* PostgresQL
* Devise
* Foundation
* Sass
