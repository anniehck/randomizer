# Randomizer

#### Bored? Need something to watch, read, look at? Just randomize!

======

### Set up

git clone `https://github.com/anniehck/randomizer.git`

bundle install

npm install

rake db:create

rake db:migrate

rails server

npm start

### Technologies

* Ruby 2.3.1
* React.js
* Rails 4.2.6
* Devise
* Foundation
* Sass

### API Sources

* NASA
* Movie DB
* Beer DB
* Astrocast
* Tacofancy
